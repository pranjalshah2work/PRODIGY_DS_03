# PRODIGY_DS_03

## Task Objective
Build a Decision Tree Classifier to predict whether a customer will purchase a product or service based on customer data.

## Tools & Technologies Used
- Python
- Jupyter Notebook
- Pandas
- Scikit-learn
- Matplotlib
- GitHub

## Dataset Used
Bank Marketing Dataset

## Steps Performed
1. Loaded and explored the dataset
2. Selected important features:
   - age
   - balance
   - duration
   - campaign
3. Converted target values (yes/no) into numerical values
4. Split dataset into training and testing data
5. Trained a Decision Tree Classifier
6. Predicted customer responses
7. Evaluated model accuracy
8. Visualized the Decision Tree

## Model Accuracy
The model achieved approximately 83% accuracy.

## Files Included
- Jupyter Notebook (.ipynb)
- Dataset (.csv)
- Decision Tree Visualization (.png)

## Outcome
Successfully built and evaluated a Decision Tree Classifier model for customer purchase prediction.
